# wtz

_What timezone?_

A small command line tool that shows times in timezones you're interested in.

Your computers local timezone and UTC:
```
$ wtz --timezones UTC 
+-----------+-------+
| MELBOURNE |  UTC  |
+-----------+-------+
| 00:00     | 14:00 |
| 01:00     | 15:00 |
| 02:00     | 16:00 |
| 03:00     | 17:00 |
| 04:00     | 18:00 |
| 05:00     | 19:00 |
| 06:00     | 20:00 |
| 07:00     | 21:00 |
| 08:00     | 22:00 |
| 09:00     | 23:00 |
| 10:00     | 00:00 |
| 11:00     | 01:00 |
| 12:00     | 02:00 |
| 13:00     | 03:00 |
| 14:00     | 04:00 |
| 15:00     | 05:00 |
| 16:00     | 06:00 |
| 17:00     | 07:00 |
| 18:00     | 08:00 |
| 19:00     | 09:00 |
| 20:00     | 10:00 |
| 21:00     | 11:00 |
| 22:00     | 12:00 |
| 23:00     | 13:00 |
+-----------+-------+
```

Or other timezones, not including your local timezone, with the first timezone listed being the main timezone starting at `00:00`:
